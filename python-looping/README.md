# Python Looping Samples (IB Computer Science)

Sample scripts demonstrating Python loop constructs mapped to IB CS pseudocode.

| File | Topic |
|---|---|
| `01_for_loop_range.py` | Count-controlled `for` loop with `range()` |
| `02_for_loop_collection.py` | `for` loop iterating over a list/string, `enumerate()` |
| `03_while_loop.py` | Condition-controlled `while` loop |
| `04_repeat_until.py` | REPEAT-UNTIL pattern using `while True` + `break` |
| `05_nested_loops.py` | Nested loops, multiplication table, 2D grid processing |
| `06_break_and_continue.py` | `break` and `continue` statements |
| `07_sentinel_and_accumulator.py` | Sentinel-controlled loops and accumulator pattern |

Run any file directly, e.g.:

```bash
python3 01_for_loop_range.py
```
